Política de Privacidad de ALERTA

Última actualización: 24 de marzo de 2026

Esta Política de Privacidad describe cómo la aplicación ALERTA (paquete: com.example.alerta) recopila, usa, almacena y protege la información cuando usted utiliza nuestros servicios.

Nota sobre esta versión: esta es una versión de prueba para testers. No incluye compras dentro de la aplicación ni procesamiento de pagos.

2. Qué hace la app

ALERTA es una aplicación para dispositivos Android que monitoriza eventos del sistema y reproduce alertas por voz (texto a voz / TTS) cuando detecta cambios relevantes, como conexión Wi-Fi, datos móviles, estado del cargador o batería, temperatura, uso de memoria RAM, espacio de almacenamiento u otros eventos configurados por el usuario.

3. Datos que utiliza la aplicación

ALERTA está diseñada para funcionar principalmente de forma local en el dispositivo del usuario. La App puede acceder a información del dispositivo necesaria para su funcionamiento, incluyendo de forma no exhaustiva:

  • Estado de conectividad (Wi-Fi, datos móviles, Ethernet u otros tipos de red).
  • Estado de carga y nivel de batería.
  • Información térmica del dispositivo (temperatura de núcleos del procesador y/o batería, cuando el sistema lo permite).
  • Uso de memoria RAM y espacio libre de almacenamiento interno.
  • Consumo de datos móviles (para alertas de uso excesivo, si el usuario las activa).
  • Datos del acelerómetro (solo para detectar agitación del dispositivo y pausar temporalmente las alertas de voz, si el usuario activa esta función).
  • Configuraciones elegidas por el usuario (idioma, tipo de alertas, umbrales, voces TTS, alertas programadas, etc.).

Esta información se procesa en el dispositivo para generar alertas y notificaciones. ALERTA no vende datos personales ni los utiliza con fines publicitarios.

4. Datos y permisos utilizados

Para ofrecer sus funciones, ALERTA puede solicitar permisos del sistema Android, tales como:

  • Conectividad de red y Wi-Fi (para detectar cambios de conexión).
  • Servicio en primer plano (para mantener el monitoreo activo con notificación visible).
  • Notificaciones (para avisos al usuario).
  • Audio y vibración (para alertas por voz y aviso háptico).
  • Inicio automático tras reinicio del dispositivo (si el usuario lo habilita).
  • Optimización de batería (para solicitar al usuario excluir la App de restricciones, si es necesario para el monitoreo en segundo plano).
  • Almacenamiento (solo si fuera necesario por compatibilidad del sistema o versión de Android).

ALERTA no utiliza la ubicación GPS para rastrear al usuario ni para crear perfiles de localización. Si algún permiso relacionado con ubicación o teléfono aparece solicitado por el sistema, se usa únicamente con fines técnicos de compatibilidad del dispositivo y no para identificar al usuario ni para seguimiento.

5. Servicio en primer plano

ALERTA utiliza un servicio en primer plano para mantener activo el monitoreo en segundo plano y ofrecer alertas oportunas. El usuario verá una notificación persistente mientras el servicio esté activo. Esto es necesario para el funcionamiento principal de la App.

6. Almacenamiento local

Las preferencias del usuario y la configuración de la App se guardan localmente en el dispositivo (por ejemplo, mediante almacenamiento interno del sistema / SharedPreferences). En esta versión de prueba no se gestionan compras ni pagos dentro de la aplicación.

7. Texto a voz (TTS)

Las alertas por voz utilizan el motor de texto a voz (TTS) instalado en el dispositivo del usuario (por ejemplo, servicios del sistema o de Google). ALERTA no envía a servidores propios el contenido completo de las alertas para su reproducción; la síntesis de voz se realiza mediante el motor TTS del dispositivo o del proveedor correspondiente según la configuración del sistema.

8. Recopilación y uso de datos personales

  • ALERTA no recopila, vende ni comparte datos personales de forma remota con fines comerciales o publicitarios.
  • ALERTA no utiliza datos personales para fines publicitarios.
  • Toda la configuración y los datos de monitoreo se procesan y almacenan localmente en el dispositivo, salvo los procesados por servicios del sistema cuando el usuario utiliza funciones técnicas del dispositivo (TTS, notificaciones, red, etc.).

9. Compartición con terceros

ALERTA no comparte datos personales con terceros con fines comerciales o publicitarios. La App puede interactuar con servicios del sistema y, en su caso, con Google Play para distribución e instalación de la App y funciones técnicas del dispositivo (TTS, notificaciones, red, etc.).

10. Transferencias internacionales

ALERTA no opera servidores propios para recopilar datos personales de los usuarios. Si el usuario utiliza servicios de Google (Play Store, TTS, etc.), dichos servicios pueden estar sujetos a las políticas de privacidad de Google.

11. Seguridad

Se aplican medidas razonables para proteger la información procesada localmente en el dispositivo. Dado que la App no transmite datos personales a servidores propios del desarrollador, el riesgo de exposición remota de datos por parte de ALERTA es limitado.

12. Conservación de datos

La configuración y los datos locales se conservan en el dispositivo hasta que el usuario los modifica, desinstala la App o borra los datos de la aplicación desde los ajustes del sistema.

13. Derechos del usuario

El usuario puede:

  • Desactivar alertas y el servicio desde la configuración de la App.
  • Revocar permisos desde los ajustes de Android.
  • Desinstalar la App en cualquier momento, lo que elimina los datos locales asociados a la aplicación (salvo copias de seguridad del sistema, si las hubiera).
